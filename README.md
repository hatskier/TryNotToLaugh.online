# TryNotToLaugh.online

Web game built on blockstackm where you watch funny videos and try not to laugh.
- Ads-free
- Secure
- Decentralised
- AI emotions analyzer

## AI emotions analyzer
It works directly in browser

## To develop
```bash
npm i
npm run serve
```

## To build for production
```bash
npm i
npm run build
```
